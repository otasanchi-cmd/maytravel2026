スポット案内

Spots & Highlights
淡路島
淡路ワールドパーク ONOKORO
淡路島中部に位置するテーマパーク。世界の名所をミニチュアで再現したワールドパビリオンや、ジップライン、観覧車など。平日10:00〜17:00。大人¥1,600。
→ 公式サイト
南淡路
うずしおクルーズ
大鳴門橋の下で生まれる世界最大級の渦潮を間近で体験できる遊覧船。約1時間のクルーズ。出航時刻は潮によって変動。大人¥2,500。9:30便を予約済。
→ 公式サイト・時刻表
鳴門（徳島）
大塚国際美術館
鳴門市に位置する世界最大級の常設展示の美術館。西洋名画1,000余点を陶板で原寸大に再現。鑑賞ルート約4km。9:30〜17:00（入館〜16:00）、月曜休館。大人¥3,300。
→ 公式サイト
淡路島北部
あわじ花さじき
淡路島北部の丘陵地に広がる広大な花畑。明石海峡と神戸の街並みを望む絶景と、四季折々の花が楽しめる。入園無料。5月はポピーやネモフィラが見頃。
→ 公式サイト
彦根
国宝 彦根城
江戸時代初期に築かれた国宝の天守。全国に現存する天守の中でも特に保存状態がよく、城郭全体が国の特別史跡。8:30〜17:00（最終入場16:30）。大人¥800。
→ 公式サイト
近江八幡
安土城跡
織田信長が築いた幻の名城跡。天守台石垣など遺構が残り、信長の壮大な夢を感じられる。近江八幡市安土町。8:30〜16:00。大人¥700。
→ 公式サイト
近江八幡
La Collina 近江八幡
たねやグループが手がける茅葺き屋根の複合施設。バームクーヘンや和洋菓子が楽しめる。広大な自然の中でゆったりとした時間を過ごせる人気スポット。9:00〜18:00。
→ 公式サイト
長浜
黒壁スクエア
長浜市の旧市街地に広がるガラス工芸を中心とした観光エリア。黒漆喰の蔵と洋館が並ぶレトロな街並みが魅力。ガラスショップ、カフェ、雑貨店が集まる。
→ 公式サイト
関ヶ原（岐阜）
岐阜関ヶ原古戦場記念館
1600年の天下分け目の関ヶ原の戦いを伝える記念館。映像シアターや武将の甲冑、戦況ジオラマなど充実の展示。古戦場散策との合わせがおすすめ。9:30〜17:00。
→ 公式サイト
